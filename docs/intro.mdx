# 💎 Welcome to Crystal Studio Guides

Welcome to the official learning portal of the **Crystal Studio Community**! 🚀

Our goal is to help you learn software engineering, collaborate on real-world projects, and master the tools of the trade.

---

## 🗺️ Learning Directory

Here is our structured catalog of guides to help you build your programming foundation:

### 1️⃣ Version Control & Onboarding
* 💻 **[Git & GitHub Basics](git-basics.md)**: Forking, cloning, committing, and opening Pull Requests.
* 🔓 **[Membership Profile Visibility](profile-visibility.md)**: Displaying the organization badge on your GitHub profile.
* ✍️ **[Blogging Guide](blogging-guide.md)**: Learn how to publish posts and write articles on this website.

### 2️⃣ Programming Languages
* 🐍 **[Python Basics](python-basics.md)**: Variables, conditional structures, collections, and pip environments.
* ☕ **[Java Basics](java-basics.md)**: OOP principles, classes, methods, compiling, and running Java programs.
* ⚙️ **[C Programming](c-programming.md)**: Compilation with GCC, memory management, and pointers.
* 🚀 **[C++ Programming](cpp-programming.md)**: Standard Template Library (STL), memory, and modern smart pointers.

### 3️⃣ Web Development & Databases
* 📊 **[SQL & Databases](sql-basics.md)**: SELECT statements, aggregations, and combines using Joins.
* 🎨 **[HTML & CSS Basics](html-css-basics.md)**: Semantic formatting, Box Model, and Flexbox/Grid systems.
* ⚡ **[JavaScript Basics](javascript-basics.md)**: Document manipulation (DOM), array mapping, and Async/Await requests.
* 💎 **[TypeScript Basics](typescript-basics.md)**: Explicit annotations, interfaces, union types, and type checks.

### 4️⃣ Documentation
* 📝 **[Markdown Basics](markdown-basics.md)**: Formatting readmes, list checklists, tables, and alert admonitions.
